# Delivery Resources

Delivery is the transmission of the weapon to the targeted environment.

## Phishing Tools

*   **[o365-attack-toolkit](https://github.com/mdsecactivebreach/o365-attack-toolkit)**: A toolkit to attack Office365.
*   **[Evilginx2](https://github.com/kgretzky/evilginx2)**: Man-in-the-middle attack framework used for phishing credentials and session cookies.
*   **[Gophish](https://github.com/gophish/gophish)**: Open-source phishing toolkit designed for businesses and penetration testers.
*   **[PwnAuth](https://github.com/fireeye/PwnAuth)**: Web application framework for launching and managing OAuth abuse campaigns.
*   **[Modlishka](https://github.com/drk1wi/Modlishka)**: Flexible and powerful reverse proxy for ethical phishing campaigns.

## Watering Hole Attack Tools

*   **[BeEF](https://github.com/beefproject/beef)**: The Browser Exploitation Framework. Focuses on the web browser.
