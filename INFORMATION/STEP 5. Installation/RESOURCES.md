# Installation Resources

Installation allows the adversary to maintain persistence inside the environment.

## Persistence Tools

*   **[SharpStay](https://github.com/0xthirteen/SharpStay)**: .NET project for installing Persistence.
*   **[SharPersist](https://github.com/fireeye/SharPersist)**: Windows persistence toolkit written in C#.
*   **[SharpHide](https://github.com/outflanknl/SharpHide)**: Tool to create hidden registry keys.
*   **[DoUCMe](https://github.com/Ben0xA/DoUCMe)**: Leverages the NetUserAdd Win32 API to create a new computer account.
*   **[ABPTTS](https://github.com/nccgroup/ABPTTS)**: TCP tunneling over HTTP for web application servers.
*   **[pivotnacci](https://github.com/blackarrowsec/pivotnacci)**: A tool to make socks connections through HTTP agents.
*   **[reGeorg](https://github.com/sensepost/reGeorg)**: The successor to reDuh, pwn a bastion webserver and create SOCKS proxies.
*   **[DAMP](https://github.com/HarmJ0y/DAMP)**: The Discretionary ACL Modification Project: Persistence Through Host-based Security Descriptor Modification.
*   **[IIS-Raid](https://github.com/0x09AL/IIS-Raid)**: A native backdoor module for Microsoft IIS.
*   **[SharPyShell](https://github.com/antonioCoco/SharPyShell)**: Tiny and obfuscated ASP.NET webshell for C# web applications.
*   **[ScheduleRunner](https://github.com/netero1010/ScheduleRunner)**: C# tool to customize scheduled task for persistence.
*   **[SharpEventPersist](https://github.com/improsec/SharpEventPersist)**: Persistence by writing/reading shellcode from Event Log.
*   **[Kraken](https://github.com/kraken-ng/Kraken)**: Modular multi-language webshell.
*   **[HiddenDesktop](https://github.com/WKL-Sec/HiddenDesktop)**: HVNC for Cobalt Strike.
